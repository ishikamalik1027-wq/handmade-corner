Project Name: Artisan's Corner

Description:
This is a handmade products marketplace where artisans can list and sell their products.

Technologies Used:
Node.js
Express.js
MongoDB
EJS
CSS

How to Run:
1. Download the project
2. Open terminal in project folder
3. Run command: npm install
4. Run command: npm start or nodemon app.js
5. Open browser: http://localhost:8080
